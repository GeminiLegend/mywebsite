/*
what is a parameter

parameters are used when defining a function.

what is an argument

arguments are used to invoke a function

you give a function a parameter when you want to 
make it reusable and dynamic. this allows you to
pass arguments to the function and apply the same
function to different sets of data
*/

function multiply (x, y) {
	return x * y;
};

multiply(12, 3);

// would return 36

multiply(10, 4);

// would return 40
